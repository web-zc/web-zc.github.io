this指向
javascript
前端
发布日期:   2019-04-20

最近看了几篇有关this的博客总结下

前言：

this指向一直令人头疼，每当自己快弄懂得时候，又来些奇怪的案例，一战回到解放前。

第一次写博客，谈一下现在对this的理解。

如果有不对的地方，欢迎大家指正。

一、this指向

  1. 普通函数调用 this是全局对象window 这个函数实质是window下的方法

    ```javascript

    var x = 520
    function test(){

    console.log(this.x) // 520

    }
    test()

    ```

  2. 函数作为对象的方法被调用 函数的this就是调用它的对象

    ```javascript

    function test(){

            console.log(this.x)
    }
    var obj = {}
    obj.x = 1
    obj.m = test
    obj.m() // 1

   ```

  1. 构造函数 构造函数中this是实例化的对象

    ```javascript

    function test(){

           this.x = 520
   }
   var obj = new test()
   obj.x  //520
```

  2. 箭头函数 箭头函数的this是，上一作用域的this 且无法通过call方法改变this指向

    ```javascript

    var arry = [1]

       let x = 520
   function test(){
      arry.forEach(itme => {
          console.log(this.x) // 520
      })
   }
   test()

```

    对象中箭头函数 对象中箭头函数的this指向window

    ```javascript

    var x = 520

      var obj = {
      x:1314,
      y: ()=>{
          console.log(this.x) // 520
      },
      z: {
         x: 1314,
         d: ()=>{
             console.log(this.x) // 520
         }
      }
  }
  obj.y() // 对象中箭头函数this是window
  obj.z.d() // 多层对象嵌套中的箭头函数的this还是window
```

  1. 原型方法 原型方法中的this是实例对象

    ```javascript

    function Test(){

          this.x = 520
   }
   Test.prototype.test = function(){
        console.log(this.x) //520
   }
   var foo = new Test()
   foo.test()
```

  2. 匿名函数 匿名函数中的this是window 匿名函数会自执行

    ```javascript

    var x =520

       function Test(){
      var x = 1314
     return function (){
          console.log(this.x)  // 520
      }

   }
      var z = new Test()
      z()

```

二、改变this指向的方法

  • call、apply、bind都属于Function.prototype的一个方法，所以每个function实例都有call、apply、bind属性

    ```javascript

    var obj = {

         b: function(x,y){
         console.log(this)
         console.log(x,y)
     }
 }
 //以下所有调用都可以传递参数
 obj.b(2,4) //obj
 obj.b.call(window,2,4) // window call与apply 都会返回新函数
 obj.b.apply(window,[2,4]) // window
 obj.b.bind(window,2,4)() // window bind返回改变了上下文后的一个函数，要执行，还要调用
```

  • 用变量或对象下的方法接收，会改变this指向

    ```javascript

    var x = 1314

        var obj = {
        x: 520,
        b: function () {
            console.log(this.x)
        }
    }
    var c = {
        x: 666
    }
    obj.b()  //520  
    var q = obj.b  //用普通变量接收，this是window
    q() // 1314
    c.o = obj.b    //用对象下的一个方法接收this是该对象c
    c.o() // 666
```
